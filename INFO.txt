
Необходимо написать приложение, получающее показывающее данные, которые получает с помощью запроса по URL http://private-db05-jsontest111.apiary-mock.com/androids

Приложение должно содержать два экрана:

1. Основной экран содержит: кнопку «обновить» и список показывающий значения полей title объектов. По нажатию на элемент списка открывается второй экран.

2. Вспомогательный экран показывает картинку полученную по адресу из поля img объекта и название объекта. Должно поддерживаться переключение между элементами жестом листания

Приложение должно корректно обрабатывать сворачивание и разворачивание на нажатию на иконку в лаунчере, а так же поворот экрана.

 

Приложение должно содержать кастомизированную тему оформления. Внешний вид на усмотрение разработчика


При реализации можно использовать любые библиотеки.
